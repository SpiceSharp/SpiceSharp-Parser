Class SpiceExpressionParser

@author: Sven Boulanger @author: Marcin Gołębiowski (custom functions, lazy evaluation) A very light-weight and fast expression parser made for parsing SPICE expressions It is based on Dijkstra's Shunting Yard algorithm. It is very fast for parsing expressions only once. The parser is also not very expressive for errors, so only use it for relatively simple expressions.

Supported operators

  • Positive and negative ('+', '-')
  • Addition and subtraction ('+', '-')
  • Multiplication and division ('*', '/')
  • Modulo ('%')
  • Logical ('&&', '||', '!')
  • Relational ('==', '!=', '<', '>', '<=', '>=')
