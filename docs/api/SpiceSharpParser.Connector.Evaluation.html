SpiceExpression

@author: Sven Boulanger @author: Marcin Gołębiowski (user functions) A very light-weight and fast expression parser made for parsing Spice expressions It is based on Dijkstra's Shunting Yard algorithm. It is very fast for parsing expressions only once. The parser is also not very expressive for errors, so only use it for relatively simple expressions.

Supported operators

  • Positive and negative ('+', '-')
  • Addition and subtraction ('+', '-')
  • Multiplication and division ('*', '/')
  • Modulo ('%')
  • Logical ('&&', '||', '!')
  • Relational ('==', '!=', '<', '>', '<=', '>=')

Supported functions

  • Minimum and maximum ('min(a,b)', 'max(a,b)')
  • Square root ('sqrt(a)')
  • Absolute value ('abs(a)')
  • Exponent and logarithms ('exp(a)', 'log(a)', 'log10(a)')
  • Powers ('pow(a, b)')
  • Trigonometry ('sin(a)', 'cos(a)', 'tan(a)', 'asin(a)', 'acos(a)', 'atan(a)', 'sinh(a)', 'cosh(a)', 'tanh(a)', 'atan2(a, b)')
